# Stardust - Приложение для знакомств

## 1. Обзор проекта

**Название:** Stardust  
**Тип:** Кроссплатформенное приложение для знакомств (Web + Android + iOS)  
**Технология:** Flutter (единая кодовая база)  
**Стилистика:** Космос, фиолетово-синие тона, звездное небо

## 2. UI/UX Спецификация

### Цветовая палитра
- **Основной:** `#6B4EFF` (космический фиолетовый)
- **Вторичный:** `#1A1A2E` (глубокий космос)
- **Акцент:** `#00D9FF` (неоновый голубой)
- **Фон:** `#0F0F1A` (темный космос)
- **Поверхность:** `#1E1E32` (карточки)
- **Текст основной:** `#FFFFFF`
- **Текст вторичный:** `#A0A0B0`
- **Успех:** `#00FF88`
- **Ошибка:** `#FF4F6D`

### Типографика
- **Заголовки:** Montserrat Bold
- **Подзаголовки:** Montserrat SemiBold
- **Основной текст:** Inter Regular
- **Кнопки:** Inter SemiBold

### Компоненты UI
- Звездный фон с анимацией (для главной и splash)
- Карточки профилей с фото и основной инфой
- Космические кнопки с градиентом
- Плавающая навигация (bottom navigation)
- Модальные окна с размытием фона

### Навигация (мобильное приложение)
1. **Главная** (поиск анкет)
2. **Лайки** (кто лайкнул)
3. **Сообщения**
4. **Профиль**

### Навигация (web - главное меню)
- Главная (Landing)
- Правила
- Документация
- Политика
- Вход/Регистрация → Приложение

## 3. Функциональная спецификация

### Экраны приложения

#### 1. Поиск анкет (Главная)
- Свайп-карточки анкет (фото, имя, возраст, город)
- Кнопки: лайк (сердце), дизлайк (крестик), суперлайк (звезда)
- Фильтры поиска (возраст, пол, расстояние)

#### 2. Кто лайкнул
- Список пользователей, которые лайкнули
- Счетчик лайков
- Быстрый переход к анкете

#### 3. Сообщения
- Список чатов
- Предпросмотр последнего сообщения
- Индикатор непрочитанных
- Экран чата (заглушка)

#### 4. Профиль пользователя
- Фото профиля (загружаемое)
- Имя, возраст, город
- О себе
- Интересы/теги
- Настройки анкеты (что видят другие)
- Кнопка "Редактировать"

#### 5. Настройки профиля
- Личные данные
- Настройки поиска (пол, возраст, расстояние)
- Уведомления
- Приватность
- Выйти

#### 6. Регистрация/Вход
- Email + пароль
- Соцсети (заглушки)
- Валидация полей

### Экраны сайта (web)

#### 1. Главная (Landing)
- Логотип Stardust
- Описание приложения
- Призыв к действию (скачать/зарегистрироваться)
- Скриншоты приложения
- Звездный анимированный фон

#### 2. Правила площадки
- Текстовый контент с правилами

#### 3. Документация
- API документация (базовая структура)

#### 4. Политика сайта
- Политика конфиденциальности

#### 5. Правила
- Дублирование правил площадки
